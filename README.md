# QuickNotes
A simple Notes App built with Python.  
Users can add, view, and delete notes using a terminal-based menu system.

---
# Features
- Add notes
- View notes
- Delete notes
- Simple menu interface
- Beginner-friendly project

---
# Technologies Use
- Python

---
# Concepts Used
This project uses:
- Lists
- Functions
- Loops
- Conditions
- Exception Handling
- User Input

---
# Installation
1. Install Python:
https://www.python.org/

2. Download or clone the repository.

3. Open terminal or command prompt.

4. Run the Python file.

---
# How to Run
```bash
python main.py
```
# How It Works
1. User selects an option from the menu.
2. Notes can be added.
3. Saved notes can be viewed.
4. Notes can also be deleted.
5. Program continues until user exits.
